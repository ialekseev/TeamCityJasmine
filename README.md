TeamCityJasmine
===============

TeamCity + JasmineJs + PhantomJs standalone example.


TeamCity command to execute: *phantomjs.exe test_runner.js*

To test locally: *run_local.bat*

Notice:
test_runner.js should has valid path to TestRunner.html
