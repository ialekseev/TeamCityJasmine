TeamCityJasmine
===============

TeamCity + JasmineJs + PhantomJs standalone example.


TeamCity command to execute: phantomjs.exe spec_runner.js

To test locally: run_local.bat

Notice:
spec_runner.js should has valid path to SpecRunner.html
